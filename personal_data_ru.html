1. Объем и цели обработки

При создании аккаунта в игре "Imposter 3D: online horror" я предоставляю и разрешаю обрабатывать только следующие данные, необходимые для работы игры:

  • Логин (никнейм) для идентификации в игре
  • Базовые данные аутентификации через Firebase Authentication для защиты аккаунта
  • Игровой прогресс: текущий уровень, игровая валюта и прочие внутреигровые предметы
  • Технические данные для работы игры: время последнего входа и версия игры

Эти данные используются исключительно для:

  • Создания и защиты игрового аккаунта
  • Сохранения игрового прогресса
  • Обеспечения корректной работы игры
  • Предоставления технической поддержки

2. Учетная запись

При создании учетной записи вы соглашаетесь:

  • Предоставлять точную и актуальную информацию
  • Нести ответственность за все действия, совершенные через вашу учетную запись

3. Передача данных

Оператор использует только необходимые сервисы Google Firebase для базового функционирования игры. Мы не продаем и не передаем ваши данные третьим лицам, за исключением случаев, предусмотренных законодательством.

4. Срок хранения данных

Данные хранятся только на время существования аккаунта. При удалении аккаунта все данные уничтожаются в течение 30 дней, за исключением информации, которая должна храниться по закону.

5. Права пользователя

Вы имеете право:

  • Получить информацию о ваших данных
  • Исправить неточные данные
  • Удалить ваш аккаунт и все связанные данные
  • Отозвать согласие на обработку данных

6. Удаление аккаунта

Для удаления аккаунта и всех данных необходимо:

  1. Открыть меню "Онлайн"
  2. Выбрать "Аккаунт"
  3. Выбрать "Настройки аккаунта"
  4. Нажать "Удалить аккаунт"
  5. Подтвердить удаление
